### 📦 Installation
```bash
git clone https://github.com/your-repo.git
cd your-repo
npm install
```

### 🏃‍♂️ Running Locally
```bash
npm run dev
```
Then, open [http://localhost:5173/](http://localhost:5173/) in your browser.

### 🏗️ Build for Production
```bash
npm run build
```
